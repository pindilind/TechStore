# Techstore
Skolarbete, Techstore

TechStore är en e-handelsbutik som säljer telefoner av kvalité.
Det finns möjlighet att registrera och logga in på din sida för att
se dina tidgare beställningar. 


VG-krav 
Vi valde att göra en popup-meny för att behålla den vanliga layouten på sidan. 
Genom att lägga till en icon i headern där man klickar fram menyn. I menyn kan 
man även öppna login/register formen. Som också är en popup som går att stänga på 
startsidan.
Anledningen till att vi gjorde detta var för att slippa använda oss utav fler html-filer.
Vi använde oss av samma färger för att hålla det enhetligt. 

Menyn kan också ta dig till där du kan se dina beställningar alltså orderhistorik. 
Här blev det en väldigt simpel layout.
